# Chat Anonymous Private.
  Link : https://chat-anonymous-private.netlify.app/

This repository contains the static HTML page for the Privacy Policy of the **Chat Anonymous Private** website. The site is designed to provide users with a secure and anonymous chat experience. This document outlines our approach to data collection, encryption, group message visibility, and other key privacy practices.

## Features

- **State-of-the-Art Encryption:** Data is encrypted during transmission to ensure that messages are securely sent between your device and our servers.
- **Group Message Accessibility:** Messages sent within a group are visible to all members of that group.
- **Responsive Design:** The privacy policy page is designed to work seamlessly across desktops, tablets, and mobile devices.
- **Clear Policy Outline:** Detailed sections covering information collection, encryption and data security, message visibility, data retention, and contact information.
